## Tarea
Cada integrante debe:
1. Hacer un fork del repositorio del líder.
2. Clonar su fork en su computadora.
3. Crear una rama con su nombre.
4. Editar el README del equipo añadiendo su nombre.
5. Hacer commit y push.
6. Enviar un Pull Request al repositorio del líder.

El líder debe:
1. Aceptar los Pull Requests.
2. Consolidar los cambios.
3. Enviar un Pull Request al repositorio principal (profesora).
